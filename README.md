# WarpVoyage

Immersive web-based universe travel simulator. Choose destinations, experience first-person warp journeys, and explore a 3D universe map.

## Features
- 3D Interactive Universe Map
- First-Person Warp Travel
- Educational Content on Real Exoplanets
- AI-Generated Stories
- Admin Dashboard

## Tech Stack
- React + Vite
- Three.js + React Three Fiber
- Tailwind CSS
- TypeScript

## Getting Started

```bash
npm install
npm run dev
```
